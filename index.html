TP Cfun


Actuellement, le complexe nommé « C Fun », ouvert de 6h à 22h tous les jours, dispose de 2 salles distinctes selon le type d’équipement. Il possède 4 équipements type musculation et 5 équipements de type fitness. Le but du TP est de réaliser une application javafx Maven graphique reliée à une base de données. Cette application doit permettre l’authentification d’employés. Cette application permet à l’employé connecté de voir les places restantes dans chacun des espaces et d’y entrer s’il y a de la place et ensuite d’y sortir.
